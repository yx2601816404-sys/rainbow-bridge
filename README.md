# 🌉 彩虹桥 — 每一颗星，都是一个名字

> *就在天堂的入口处，有一个叫做彩虹桥的地方。当一只和人类有过特别羁绊的动物离开这个世界，他会去到彩虹桥。*

**[✦ 打开彩虹桥](https://yx2601816404-sys.github.io/rainbow-bridge/)**

彩虹桥是一个为宠物建造的在线纪念空间。每一只被记住的毛孩子，都会成为星空中的一颗星。

## 它是什么

一片星空。每颗星代表一只被纪念的宠物。

你可以：
- 🌟 为离开的毛孩子**点亮一颗星**
- 🕯️ 为他们**点蜡烛**、写寄语
- 💬 在留言墙写下思念
- 🔗 分享纪念馆链接给朋友

## 为什么做这个

2024 年 10 月 20 日，一只叫小谦的布偶猫离开了这个世界。他来了 19 个月，然后安静地走了。

彩虹桥的第一颗星，亮给小谦。

## 设计理念

一个失去宠物的人，深夜独自打开这个页面。他需要感受到什么？

不是被技术打动。是被安静地接住。

- 开屏是一颗星慢慢亮起，不是炫技动画
- 创建纪念馆是仪式，不是填表
- 深夜的星空更亮，凌晨有一丝暖色
- 零广告，零追踪，基础纪念永远免费

## 技术

纯静态站。零依赖。HTML + CSS + JavaScript。

60KB 部署大小。打开即用。
