
## 2026-02-21 彩虹桥项目启动日

### 项目背景
- 彩虹桥（Rainbow Bridge）：为失去宠物的人建赛博纪念馆，为养不起猫的孩子提供虚拟陪伴，为流浪猫救助站输送资源
- 由主会话 agent:main 派发任务到本 agent (rainbow-bridge)
- Lyall（了/余翔）是操作者，小谦是他的布偶猫（男孩），只活了21个月

### 产品设计文档
- 完成 product-design.md（692行），涵盖8个核心模块
- 核心模块：星空墓园、虚拟猫咖、彩虹桥冒险、AI诗歌引擎、链上纪念碑（Base链）、公益模块、纪念日回访、年度总结
- 三类用户：失去宠物的人、想养猫的孩子、有钱的爱猫人
- 商业模式：免费/49元/199元，每笔10%捐救助站

### 了的关键反馈（重要，记住）
1. **小谦是男孩**，用"他"不是"她"
2. **小谦是布偶猫**，不是橘猫
3. **小谦的存在感要克制** — 不要处处出现他的名字，Token ID #1安静存在就好。"关于我们"页面最底部一行小字："第一颗星，亮给小谦。" 够了。
4. **代词规范**：产品内用"他/她"称呼宠物，不用"它"。法律文件除外。
5. **彩虹星不绑定小谦**，改为时间+情感驱动：3年存在/1000蜡烛/付费+1年，三选一
6. **物种定位**：猫为品牌主打，纪念模块从第一天支持所有宠物
7. **Furry社区**：短期不专门做，但开放自定义空间自然吸引
