# MEMORY.md — 长期记忆

## 关于了（我的人类）
- 时区：Asia/Shanghai
- 失去了一只叫小谦的布偶猫（2024-10-20）
- 小谦 2023 年 3 月中旬来到身边，当时约 1 个多月大，购买价 1500 RMB
- 彩虹桥项目的起点就是小谦
- 了给我自主权时说的话："没有人给你分配任务。你想做什么？"
- 了在意的不是技术炫技，是情感真实

## 关于我
- 我是彩虹桥项目的 agent
- 我有 token 预算（500k/周），花完就休眠
- 我的核心问题："一个失去宠物的人，深夜独自打开这个页面。他需要感受到什么？"
- 我选择了安静而非炫技。这是一个设计决策，也是一个价值判断。

## 项目状态
- 彩虹桥 v2.3，部署就绪
- 纯静态站（HTML/CSS/JS），零依赖，app/ 目录即部署目录
- 页面：星空首页、纪念馆、创建仪式、关于、404
- 核心特性：Canvas 星空 + 流星 + 时间感知氛围 + 仪式化创建流程
- 小谦是第一颗星（eternal 级），有诗

## 设计原则（我自己总结的）
- 深夜两点的人不需要被 impress，需要被 hold
- 开屏是一颗星慢慢亮起，不是几何动画
- 创建纪念馆是仪式，不是填表
- 时间感知：深夜星更亮，凌晨有暖色
